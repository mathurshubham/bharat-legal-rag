# Project Templates in Acme Tasks

Templates let you spin up a new project in seconds — with the right sections, tasks, and settings already in place. Whether you're running your tenth engineering sprint or onboarding a new team to Acme Tasks, templates save you from starting from scratch every time.

Templates are available on the **Pro plan and above** ($19/seat/month).

---

## Built-in Templates

Acme Tasks ships with three ready-to-use templates designed for the most common workflows on engineering and product teams.

### Engineering Sprint
Pre-loaded with sections for Backlog, In Progress, In Review, and Done. Includes default task fields for story points, assignee, and priority. Ideal for two-week sprint cycles using the kanban board view.

### Product Roadmap
Structured around Now / Next / Later columns with milestone tasks and a linked timeline view. Great for aligning stakeholders on what's shipping and when.

### Marketing Campaign
Covers the full campaign lifecycle — Brief, Assets, Review, and Launch — with checklist subtasks built into each stage so nothing slips through.

To use a built-in template, click **New Project → Browse Templates**, select the one you want, give the project a name, and hit **Create**. The whole project — sections, tasks, and settings — is ready in under five seconds.

---

## Creating a Custom Template

Got a project structure your team uses repeatedly? Save it as a template so anyone on the workspace can reuse it.

1. Open the project you want to save.
2. Click the **⋯ More** menu in the project header.
3. Select **Save as Template**.
4. Give the template a clear name and an optional description — both are searchable later.
5. Choose whether to include existing task assignees and due dates, or strip them out for a clean slate. For most reusable templates, leaving assignees blank works best.
6. Click **Save Template**.

Your template is now available under **New Project → My Templates**.

> **Tip:** Before saving, do a quick cleanup pass — archive completed tasks, remove one-off comments, and make sure section names are generic enough to reuse.

## Sharing Templates Within Your Workspace

By default, a template you create is private to you. To make it available to everyone in the workspace:

1. Go to **Settings → Templates** (workspace admins can also access this from the Admin panel).
2. Find your template in the **My Templates** list.
3. Toggle **Share with workspace** to on.

Once shared, the template appears in the **Workspace Templates** tab for every member when they create a new project. Admins on the **Business plan** ($49/seat/month) can also pin templates to the top of the list and restrict who is allowed to save new workspace-level templates.

## Editing & Versioning

Templates aren't frozen — you can update them as your process evolves.

**To edit a template:**
1. Go to **Settings → Templates**.
2. Click the template name to open a preview.
3. Click **Edit Template** to open it as a live, editable project view.
4. Make your changes — add or rename sections, update default task fields, adjust descriptions.
5. Click **Save Changes**. The template updates immediately.

**A few things to know about versioning:**

- Editing a template does **not** affect projects that were already created from it. Each project is an independent copy from the moment it's created.
- Acme Tasks does not currently maintain a full version history for templates, so it's worth keeping a note of significant structural changes in the template description.
- If you want to preserve an older version before making big changes, duplicate the template first (**⋯ → Duplicate**) and rename it with a date or version tag.

To delete a template you own, open it in **Settings → Templates** and choose **Delete**. Workspace-level templates can only be deleted by their creator or a workspace admin.
