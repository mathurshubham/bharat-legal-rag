# Keyboard Shortcuts

Move faster through Acme Tasks without touching your mouse. These shortcuts work everywhere in the app unless noted otherwise.

> **Tip:** Press **?** at any time to open the interactive shortcuts overlay.

---

## Navigation

Use these shortcuts to move between views and items instantly.

| Shortcut | Action |
|---|---|
| `J` | Move to next task |
| `K` | Move to previous task |
| `Cmd+K` / `Ctrl+K` | Open command palette |
| `?` | Show all keyboard shortcuts |

The **command palette** (`Cmd+K`) is the fastest way to jump anywhere — switch projects, change a task's status, invite a teammate, or open settings, all from one place. Start typing to filter options.

---

## Task Actions

These shortcuts work when a task is selected or open.

| Shortcut | Action |
|---|---|
| `C` | Create a new task |
| `E` | Edit the selected task |
| `Enter` | Open task detail |
| `Backspace` / `Delete` | Delete selected task (confirmation required) |

Press **C** from anywhere in a project to open the new-task composer. The task is created in the current project and status column by default.

---

## Search

| Shortcut | Action |
|---|---|
| `/` | Focus the search bar |
| `Esc` | Clear search and return focus to the board |

Press `/` to jump straight to the search bar — no need to click. Search scans task titles, descriptions, and assignees across all projects you have access to.

## Full Reference

A consolidated view of every shortcut in one place.

| Shortcut | Action | Context |
|---|---|---|
| `Cmd+K` / `Ctrl+K` | Open command palette | Global |

> **Availability:** Keyboard shortcuts are available on the **Pro, Business, and Enterprise** plans. Free plan users can use `Cmd+K`, `/`, and `?` — full shortcut support unlocks at Pro ($19/month).
